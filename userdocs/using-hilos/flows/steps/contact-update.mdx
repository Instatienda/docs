# Contact update step

The contact update step is essential in the flow builder. It enables you to save data to the contact record that you can later use to personalize the conversation and see these attributes on the [Contact tab](contact-tab) in the conversation detail in your Inbox.

You can add or update contact fields such as:

- First name
- Last name
- Email
- External URL
- [Custom attributes](create-a-contact)
