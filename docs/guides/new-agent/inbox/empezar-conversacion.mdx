---
title: 'Cómo empezar una conversación'
---
Para empezar una conversación en Hilos, primero debes navegar a la sección de Plantillas para crear o elegir una plantilla inicial. Luego, ve al Inbox y **usa el botón "+"** para abrir una nueva conversación, buscando al contacto o ingresando el número de teléfono. 

Selecciona una plantilla para comenzar la conversación y, si contiene variables, completa los valores requeridos. Espera a que el contacto responda para activar la ventana de conversación de 24 horas, permitiéndote interactuar libremente como lo harías en WhatsApp Web.
