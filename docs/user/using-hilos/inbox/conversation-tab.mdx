---
title: Conversation tab
---

The Conversation tab is a place where you can see all the information regarding the current conversation. You can see the conversation tags, the assigned agents, and the conversation history with this contact.

![Conversation tab](https://hilos-media.s3.us-east-2.amazonaws.com/documentation/userdocs/using-hilos/reference/inbox/conversation-tab/img1.png)

# Conversation tags

The conversation tags are the tags that are assigned to the current conversation. You can add or remove tags from the conversation by clicking on the + icon and searching for a tag, if it doesn't exist it will be created automatically.

![Conversation tags](https://hilos-media.s3.us-east-2.amazonaws.com/documentation/userdocs/using-hilos/reference/inbox/conversation-tab/img2.png)

# Conversation assignee

The conversation assignee is the agent that is currently assigned to the conversation. You can assign the conversation to another agent or team by clicking on the + icon next to **Assigned to**

![Conversation assignee](https://hilos-media.s3.us-east-2.amazonaws.com/documentation/userdocs/using-hilos/reference/inbox/conversation-tab/img3.png)

Then searching for an agent or team.

![Conversation assignee](https://hilos-media.s3.us-east-2.amazonaws.com/documentation/userdocs/using-hilos/reference/inbox/conversation-tab/img4.png)

# Conversation history

The conversation history is a list of all the conversation held in the past with the current contact. You can see the assignee, date, tags and template initial message.

![Conversation history](https://hilos-media.s3.us-east-2.amazonaws.com/documentation/userdocs/using-hilos/reference/inbox/conversation-tab/img5.png)
