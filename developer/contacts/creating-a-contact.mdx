## Prepare the request

To create a contact, you'll need to send a POST request to the following endpoint:

```
POST https://api.hilos.io/api/contact
```

The request body should be in JSON format, including the required and optional fields as needed. Here's a table describing each of the request data types and an example:

| Field             | Data                                                | Type             | Example                                |
| ----------------- | --------------------------------------------------- | ---------------- | -------------------------------------- |
| phone             | required                                            | string           | "555-123-4567"                         |
| first_name        |                                                     | string or null   | "John"Â                                 |
| last_name         |                                                     | string or null   | "Doe"                                  |
| email             |                                                     | string or null   | "john.doe@example.com"                 |
| meta              | object with key value pairs where value is a string | object or null   | {key1: "string", key2: "string"}       |
| external_url      |                                                     | string or null   | "http://example.com"                   |
| is_deleted        |                                                     | boolean          | false                                  |
| id                |                                                     | string uuid      | "497f6eca-6276-4993-bfeb-53cbbbba6f08" |
| source            |                                                     | string           | "website"                              |
| created_on        |                                                     | date-time        | "2019-08-24T14:15:22Z"                 |
| tags              |                                                     | Array of objects | [{"name": "customer"}]                 |
| external_id       |                                                     | string or null   | "ext-123"                              |
| default_assignees |                                                     | Array of objects | [{"id": 0}]                            |
| contact_import    |                                                     | string or null   | "cd949ab0-4614-4dda-91ec-54566ef1ac43" |

## Send the request

Using your preferred method (e.g., curl, fetch, a REST client, or a programming language library), send the prepared POST request to the endpoint.

Here's a sample request:

```json
{
  "phone": "555-123-4567",
  "first_name": "John",
  "last_name": "Doe",
  "email": "john.doe@example.com",
  "meta": {
    "property1": null,
    "property2": null
  },
  "external_url": "http://example.com",
  "is_deleted": false,
  "id": "497f6eca-6276-4993-bfeb-53cbbbba6f08",
  "source": "website",
  "created_on": "2019-08-24T14:15:22Z",
  "tags": [
    {
      "name": "customer"
    }
  ],
  "external_id": "ext-123",
  "default_assignees": [
    {
      "id": 0
    }
  ],
  "contact_import": "cd949ab0-4614-4dda-91ec-54566ef1ac43"
}
```
