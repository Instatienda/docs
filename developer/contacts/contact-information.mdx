## Prepare the request

To get contact information, you'll need to send a GET request to the following endpoint:

```
GET https://api.hilos.io/api/contact/{id}/
```

Replace `{id}` with the UUID of the contact you want to fetch.

Request parameters

- id (required): string `uuid`

## Send the request

Using your preferred method (e.g., curl, fetch, a REST client, or a programming language library), send the prepared GET request to the endpoint.

For example, if the contact's UUID is 497f6eca-6276-4993-bfeb-53cbbbba6f08, the request URL will be:

```
GET https://api.hilos.io/api/contact/497f6eca-6276-4993-bfeb-53cbbbba6f08/
```

## Handle the response

After sending the request, the API will return a response with the contact's details. The response will be in JSON format, and you can parse it as needed for your application.
